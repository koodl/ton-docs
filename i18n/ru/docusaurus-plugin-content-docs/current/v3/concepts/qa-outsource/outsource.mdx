### Astralyx

#### Summary

Компания с большим опытом работы в TON и других отраслях. Вы можете попросить нас создать дизайн, Telegram Mini Application (TMA), веб-сайт.

#### Рабочие потоки

- TON разработка смарт-контрактов (включая аудиты и тесты)
- Разработка Web 2.0 и Web 3.0
- дизайн, искусство, аренда проводов для проектов

#### Проекты

- [t.me/xjetswapbot](http://t.me/xjetswapbot) (frontend, design),
- [github.com/astralyxdev/lockup-jettons-contract](http://github.com/astralyxdev/lockup-jettons-contract) (smartcontract, web interface, tests)
- [github.com/astralyxdev/ton-proxy](http://github.com/astralyxdev/ton-proxy) (одно из первых)
- [scaleton.io](http://scaleton.io) (landing, frontend, design)

#### Контакты

[astralyx.dev](http://astralyx.dev), contact@astralyx.dev

### EvaCodes

#### Summary

EvaCodes является ведущей европейской компанией по разработке блокчейна с командами расположенных в Украине, Армении и Польше. В состав компании входят более 50 квалифицированных разработчиков, которые поставили более 60+ web3 решений, включая Web3 банковские решения, L1 блокчейны и веб-3 инфраструктуру.

#### Рабочие потоки

- DeFi
- Крипто Кошельки
- Решения на основе NFT.

#### Проекты

- [alium.finance](https://alium.finance/)

### Pixelplex

#### Summary

PixelPlex – это надежные и масштабируемые решения, работающие на блокчейне, от надежных протоколов до передовых систем управления рисками. Сфокусировавшись на достижении ощутимых результатов и повышении стоимости проекта, PixelPlex прилагает все усилия для решения сложных проблем, с которыми сталкиваются другие. Опыт компании охватывает более 450 проектов в различных отраслях, включая FinTech, здравоохранение, недвижимость и eCommerce.

#### Рабочие потоки

- Экосистемно-агностическое развитие
- Инженерия пользовательских протоколов
- Улучшение DeFi
- ZK растянуть масштабируемые решения
- Оптимизация смарт-контрактов
- Решения с поддержкой NFTP
- Предложение токена безопасности
