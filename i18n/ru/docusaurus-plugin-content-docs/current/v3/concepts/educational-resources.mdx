#### Курс TON & Telegram Blockchain

Мы рады представить **TON Blockchain Course**, который является всеобъемлющим руководством по TON Blockchain. Курс разработан для разработчиков, которые хотят научиться создавать смарт-контракты и децентрализованные приложения на TON Blockchain с помощью интерактивных способов.

Он состоит из **9 modules** и охватывает основы TON Blockchain, языка программирования FunC и TON Virtual Machine (TVM).

Кому этот курс предназначен для: инженеров по программированию, старших разработчиков Джуниор, смарт-контрактных архитекторов
