# The Open Network

**The Open Network (TON)** — децентрализованная и открытая интернет платформа, состоящая из нескольких компонентов. К их числу относятся: TON Blockchain, TON DNS, TON Storage и TON Sites. TON Blockchain является базовым протоколом, который соединяет базовую инфраструктуру TON, чтобы сформировать большую экосистему TON.

TON концентрируется на обеспечении широкого межсетевого взаимодействия при одновременном функционировании в рамках высоко масштабируемой и безопасной структуры. TON разработан для обработки миллионов транзакций в секунду (TPS) с целью в конечном итоге достичь сотен миллионов пользователей в будущем.

**TON Blockchain** разработан как распределенный суперкомпьютер или “_superserver_”, Предназначен для предоставления различных продуктов и услуг в целях содействия разработке децентрализованного видения нового Интернета.
