# Исследователи в TON

В этой статье мы рассмотрим исследователи TON, их возможности и особенности с точки зрения разработчика.

## Что такое исследователь?

Проводник - это веб-сайт, который позволяет просматривать информацию в блокчейне, такую как баланс аккаунта, история транзакций, блоки и т.д.

<Player url="https://www.youtube.com/watch?v=i5en19kzX6M" />

## Какие исследователи существуют?

Среди исследователей TON вы можете выделить несколько категорий:

- Для повседневного использования
- С расширенной информацией для разработчиков
- Специализированный

Это деление на категории в основном обусловлено, и один исследователь может относиться к нескольким категориям одновременно. Так что давайте не будем уделять слишком много внимания этому.

## Псевдоним адреса в исследователях

Сделайте адрес вашего сервиса более удобным для пользователей, используя псевдоним адресов. Создать Pull Request'ы (PR) в соответствии с представленными руководящими принципами:

- [tonkeeper/ton-assets](https://github.com/tonkeeper/ton-assets) - Tonviewer.com alias
- [catchain/address-book](https://github.com/catchain/address-book)- Tonscan.org alias

## Общая функциональность

Давайте начнем с общей функциональности, которая присутствует во всех исследователях.

Почти все исследователи имеют возможность узнать информацию о балансах, истории транзакций и информации о смарт-контракте, если он развернут по адресу.

Далее мы рассмотрим несколько исследователей, которые можно отнести к каждой из этих категорий.

## TON Scan.org

Хороший исследователь для повседневного использования. Он обеспечивает полный обзор TON Blockchain, позволяя пользователям искать транзакции, адреса, блоки и многое другое. Любой поиск производится с помощью публичной [адресной книги](https://github.com/catchain/tonscan/blob/master/src/addrbook.json) (Фонд TON Foundation, OKX и т.д.)

### Возможности

- **Удобно для повседневного использования**
- Удобно для разработчиков
- Поддержка TON DNS
- Типы контрактов
- Дизассемблер контракта

### Ссылки

- URL: https://tonscan.org/
- Testnet URL: https://testnet.tonscan.org/

## TON Scan.com

### Возможности

- Аналитика агрегации транзакций
- Отслеживание транзакций

### Ссылки

- URL: https://tonscan.com/

## Обозреватель китов

Этот проводник в большей степени ориентирован на разработчиков, чем обычные пользователи.

### Возможности

- **Удобно для разработчиков**
- Типы контрактов
- Дизассемблер контракта

### Ссылки

- URL: https://tonwhales.com/explorer

## Обозреватель Tonviewer

Этот исследователь является новейшим и имеет свои собственные уникальные возможности.
Например, Трейс. Эта функция позволяет видеть всю последовательность транзакций между смарт-контрактами, даже если последующие транзакции не содержат вашего адреса.

Информация о сделке не так подробна, как, например, на китах TON.

### Возможности

- Удобно для разработчиков
- Удобно для повседневного использования
- История транзакций в Jetton
- **Трассировка**
- Поддержка TON DNS

### Ссылки

- URL: https://tonviewer.com/
- URL Testnet: https://testnet.tonviewer.com/

## ВЫПОЛНЯЕТСЯ ТОРГОВЛЯ

Этот обозреватель специализируется на NFT, но он также может использоваться как обычный проводник.

При просмотре адреса кошелька вы можете узнать, какой NFT он хранит, и при просмотре NFT, вы можете узнать метаданные, адрес коллекции, владелец и историю транзакций.

### Возможности

- Удобно для разработчиков
- Типы контрактов
- **Специализировано в NFT**

### Ссылки

- URL-адрес: https://explorer.tonnft.tools/
- Testnet URL: https://testnet.explorer.tonnft.tools/

## ДТОН

DTON - это еще один исследователь для разработчиков. Он предоставляет много информации о сделках в удобной форме.

Также у него есть функция, позволяющая увидеть этап вычисления транзакции шаг за шагом.

### Возможности

- Удобно для разработчиков
- Расширенная информация о фазе вычисления
- Типы контрактов
- Дизассемблер контракта

### Ссылки

- URL: https://dton.io/

## TON NFTscan

Этот обозреватель специально разработан для нефункциональных токенов (NFTs) на блокчейне TON. Он позволяет пользователям изучить, отслеживать и проверять NFT транзакции, контракты и метаданные.

### Возможности

- Удобно для обычных пользователей
- Полезная информация для трейдеров, например ежедневный объем
- Рейтинг коллекции NFT

### Ссылки

- URL: https://ton.nftscan.com/

## TonStat

Отображает различные статистические данные, такие как количество зарегистрированных сетевых адресов и кошельков, объем сгорел Toncoin, объем котировок Тонкона, объем выданных НФТ, количество валидаторов и ряд других метрик.

### Возможности

- Удобно для обычных пользователей
- Полезная информация для трейдеров, например ежедневный объем
- Информация о DeFi

### Ссылки

- URL: https://www.tonstat.com/
