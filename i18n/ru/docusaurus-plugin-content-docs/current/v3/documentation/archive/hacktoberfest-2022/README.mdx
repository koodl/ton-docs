[Hacktoberfest](https://hacktoberfest.digitalocean.com/) - празднование _проектов с открытым исходным кодом_, их _сопровождающих_, а также всего сообщества _авторов_. Каждый октябрь сопровождающие с открытым исходным кодом уделяют особое внимание новым разработчикам, поскольку они направляют свои первые запросы на слияние.

Для сообщества TON пришло время помочь развитию экосистем вместе, так что давайте присоединимся к всему миру с нашим **Hack-TON-berfest** вечером и станьте экосистемой _#1 с открытым исходным кодом года_!

## Как участвовать?

Правила Хактоберфест‘19 на 2022год:

- **Хактоберфест открыт для всех**!
- Зарегистрироваться в любое время с 26 сентября по 31 октября
- Pull requests могут быть сделаны в любом GITHUB или GITLAB проекте:
  - [Список проектов по экосистеме TON](/hacktonberfest)
  - [Список проектов на GitHub](https://github.com/topics/hacktoberfest)
- Иметь **4** запросов на слияние/слияние, принятых в период с 1 октября по 31 октября
- Первые 40, 00 участников (сопровождающих и участников), которые проходят Hacktoberfest могут выбрать между двумя призами: дерево, посаженное в честь или футболку Hacktoberfest 2022. _(из сообщества Хактоберфест)_
- Каждый участник (сопровождающий и участник любого из проектов TON Ecosystem получит [**Ограниченный Hack-TON-berfest NFT**](/v3/documentation/archive/hacktoberfest-2022#what-are-the-rewards). _(из фонда TON)_

Для всех в TON это возможность стимулировать рост всей экосистемы и получить прохладную награду от фонда TON. Давайте сделаем это вместе!
