# Общий обзор

Fift является основанным на стеке языком программирования общего назначения, оптимизированным для создания, отладки и управления TON Blockchain смарт-контрактов.
Фифт разработан специально для взаимодействия с виртуальной машиной TON (VM или TVM) и TON Blockchain.

```fift
{ ."hello " } execute ."world"
hello world ok
```

:::info
Обычно использование Fift не требуется для программирования умных контрактов в TON. Однако иногда вам может понадобиться использовать фифский язык для решения необычных технических задач в рамках вашей задачи.
:::
